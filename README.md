### Procedure for idriftsættelse
 - Test på testserver
 - Kopier manuelt alle hjælpetekster til tekstdokument
 - Eksport af produkter på testserver
 - Sammenlign produktregneark imellem testserver og udvikling - lav evt. opdateringer
 - Lav eksport af produktionsdatabase og restore på testserver
 - Slet forretningsområdet på testserver
 - Genstart applikationen på testserveren så forretningsområde oprettes igen
 - Test på testserver
 - Opdatér software på produktionsserver
 - Slet forretningsområdet på produktionsserver
 - Genstart applikationen på produktionsserveren så forretningsområde oprettes igen
 - Test på produktionsserveren
 - Kopiér hjælpetekster ind på produktionsserver
